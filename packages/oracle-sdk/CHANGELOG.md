# Changelog

## [1.0.0] - 2025-11-02

### Added
- ✅ `OracleSDK` class for Oracle Consumer contracts
- ✅ `SimplePriceFeedSDK` class for Simple Price Feed contracts
- ✅ Event subscriptions with polling
- ✅ Transaction finalization helpers
- ✅ Type-safe interfaces
- ✅ Usage examples

### Features
- Read oracle status and price data
- Trigger oracle updates
- Subscribe to real-time updates
- Wait for transaction finalization
- Full TypeScript support
